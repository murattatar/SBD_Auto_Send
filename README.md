* Download [Python](https://www.python.org/downloads/) (only tested on 2.7.x)
We need these modules
* requests
* re
* time
* selenium
* win32api, win32con
* pyperclip
And [ChromeWebDriver](https://sites.google.com/a/chromium.org/chromedriver/downloads)
